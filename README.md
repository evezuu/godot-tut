# Grass Godot
I'm calling my project grass godot, and its a very simple platformer thats the result of me trying to learn how to use godot.

I made the project for Pajama Express, which is a sleepover event that I wanted to finish!

I used the Resolution platformer tutorial to guide me on how to make this. It was made using godot.

I struggled a lot with this project honestly. I had a lot of errors with file placement (at least i think it was file placement...), and at some point i even had to restart. However, i was able to finish the project in the end.

Through this project, I learned the very basics of godot!
